# 1.1 – Le réseau, c'est quoi ?

## Qu'est-ce qu'un réseau ?

Un réseau informatique c'est un ensemble de machines reliées pour échanger des données ou partager des ressources. Ce qui fait le réseau ce ne sont pas seulement les câbles ou le Wi-Fi, mais surtout les règles communes qui permettent aux équipements de se comprendre.

## Internet vs le Web

- **Internet** = réseau mondial de réseaux. Il relie des millions de réseaux plus petits répartis partout dans le monde.
- **Le Web** = un service qui fonctionne *sur* Internet, basé sur des pages accessibles via un navigateur grâce au protocole HTTP. Inventé au CERN par Tim Berners-Lee.

## Types de réseaux

| Sigle | Nom | Exemple |
|-------|-----|---------|
| PAN | Personal Area Network | Bluetooth entre un téléphone et une oreillette |
| LAN | Local Area Network | Réseau d'une maison ou d'une entreprise |
| MAN | Metropolitan Area Network | Réseau d'une ville |
| WAN | Wide Area Network | Internet |

## Équipements d'un réseau LAN

- Ordinateurs et serveurs
- Câbles Ethernet
- Cartes réseau
- Switch
- Routeur

## La commande ping

Teste la communication entre deux machines. Elle envoie des paquets ICMP et mesure le temps de réponse.

```bash
ping google.com       # accès Internet + résolution DNS
ping 192.0.2.255      # adresse de broadcast → pas de réponse
ping 172.217.20.174   # IP Google directe → timeout si filtré
```

**Lire les statistiques :**
- `perdus = 0` → connexion OK
- `perte 100%` → machine injoignable ou broadcast
