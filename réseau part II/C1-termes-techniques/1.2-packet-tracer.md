# 1.2 – Cisco Packet Tracer

## C'est quoi ?

Cisco Packet Tracer est un logiciel de simulation de réseau. Il permet de créer virtuellement des réseaux complets avec des PC, des serveurs, des switches et des routeurs, sans matériel réel.

## À quoi ça sert ?

- Tester des configurations réseau sans risque
- Visualiser le parcours des paquets de données
- Comprendre le fonctionnement des équipements réseau
- Voir immédiatement les conséquences d'une erreur de configuration

## Installation (Ubuntu)

```bash
sudo apt install ./CiscoPacketTracer_900_Ubuntu_64bit.deb
```

Se connecter avec un compte Skills for All (Cisco NetAcad).
