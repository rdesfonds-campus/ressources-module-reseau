# 1.3 – Création d'un réseau entre un PC et un serveur

## Les machines du réseau

- Un PC (PC Directeur)
- Un serveur de stockage

## Support de communication

Câble Ethernet en cuivre — les machines sont proches et le débit est suffisant.

## Identification des machines

**Adresse MAC** : adresse physique unique gravée dans la carte réseau par le constructeur. Sert à identifier une machine dans un réseau local.

**Adresse IP** : adresse logique qui identifie une machine sur un réseau. Peut être configurée manuellement ou attribuée automatiquement par DHCP.

**Masque réseau** : définit la partie réseau et la partie machine de l'adresse IP. Permet de savoir quelles machines sont sur le même réseau.

## Configuration Packet Tracer

| Machine | Adresse IP |
|---------|-----------|
| PC | 1.1.1.1 |
| Serveur de stockage | 1.1.1.2 |

## Test de communication

```
ping 1.1.1.2
```

## Parcours du paquet

PC → interface réseau → câble → Serveur → réponse sur le même chemin inverse.
